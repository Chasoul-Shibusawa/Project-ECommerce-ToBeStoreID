How To Install??
- cp .env.example .env
- Import Sql in api folder - dbs_new_tobe.sql to phpmyadmin
- composer install
- php artisan key:generate
- Copy images in public to api
- cd api
- go run main.go
- php artisan serve

Component Web.App
- Login n Register
- Dashboard
- AddProduct, Edit, And Remove / CRUD
- preview store
- Chat ( coming soon feature )
